# 🧠 BrainX – Content Aggregation Platform

BrainX is a smart content aggregation platform that collects, filters, and delivers high-quality content from multiple sources into one unified interface. It helps users stay updated with relevant information without switching between multiple apps or websites.

---

## 🚀 Features

- 🔍 **Content Aggregation**
  - Fetches data from multiple APIs, RSS feeds, and web sources
  - Supports articles, blogs, and news

- 🎯 **Personalized Feed**
  - Recommends content based on user preferences
  - Category-based filtering (Tech, Business, AI, etc.)

- ⚡ **Real-time Updates**
  - Automatically refreshes content
  - Displays latest trending topics

- 📚 **Save & Bookmark**
  - Save articles for later
  - Easy content organization

- 🔎 **Search Functionality**
  - Fast and efficient search across all content

- 📱 **Responsive Design**
  - Works across desktop, tablet, and mobile
